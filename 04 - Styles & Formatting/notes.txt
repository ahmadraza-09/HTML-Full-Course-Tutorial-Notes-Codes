游리 What is HTML Styles ?

    The HTML style attribute is used to add styles to an element, 
    such as color, font, size, and more.

    for example:

        游댖I am Red
        游댖I am Blue
        游댖I am Big

游리 The HTML Style Attribute 

    游댖Setting the style of an HTML element, can be done with the 
        style attribute.

    游댖The HTML style attribute has the following syntax:

        <tagname style="property:value;">

游리 Background Color

    The CSS background-color property defines the background color 
    for an HTML element.

    for example:

      1.<body style="background-color:powderblue;">
        <h1>This is a heading</h1>
        <p>This is a paragraph.</p>
        </body>

      2.<body>
        <h1 style="background-color:powderblue;">This is a heading</h1>
        <p style="background-color:tomato;">This is a paragraph.</p>
        </body>

游리 Text Color 

    The CSS color property defines the text color for an HTML element:

        for example:

            <h1 style="color:blue;">This is a heading</h1>
            <p style="color:red;">This is a paragraph.</p>

游리 Fonts 

    The CSS font-family property defines the font to be used for an HTML element:

    for example:

        <h1 style="font-family:verdana;">This is a heading</h1>
        <p style="font-family:courier;">This is a paragraph.</p>

游리 Text Size
    
    The CSS font-size property defines the text size for an HTML element:

        for example:

            <h1 style="font-size:300%;">This is a heading</h1>
            <p style="font-size:160%;">This is a paragraph.</p>

游리 Text Alignment
    
    The CSS text-align property defines the horizontal text alignment for 
    an HTML element:

        for example:

            <h1 style="text-align:center;">Centered Heading</h1>
            <p style="text-align:center;">Centered paragraph.</p>


游리 What is HTML Text Formatting ?

    HTML contains several elements for defining text with a special meaning.

    for example:

        This text is bold
        This text is italic
        This is subscript and superscript

游리 HTML Formatting Elements

    Formatting elements were designed to display special types of text:

    <b>      - Bold text
    <strong> - Important text
    <i>      - Italic text
    <em>     - Emphasized text
    <mark>   - Marked text
    <small>  - Smaller text
    <del>    - Deleted text
    <ins>    - Inserted text
    <sub>    - Subscript text
    <sup>    - Superscript text

游리 HTML <b> and <strong> Elements

    游댖The HTML <b> element defines bold text, without any extra importance.

        for example:

            <b>This text is bold</b>

    游댖The HTML <strong> element defines text with strong importance. 
        The content inside is typically displayed in bold.

        for example:

            <strong>This text is important!</strong>

游리 HTML <i> and <em> Elements

    游댖The HTML <i> element defines a part of text in an alternate 
        voice or mood. The content inside is typically displayed in italic.

    游댮Tip: The <i> tag is often used to indicate a technical term, 
        a phrase from another language, a thought, a ship name, etc.

        for example:

            <i>This text is italic</i>

    游댖The HTML <em> element defines emphasized text. The content 
        inside is typically displayed in italic.

    游댮Tip: A screen reader will pronounce the words in <em> with 
        an emphasis, using verbal stress.

        for example:

            <em>This text is emphasized</em>

游리 HTML <small> and <mark> Element
    
    游댖The HTML <small> element defines smaller text:

        for example:

            <small>This is some smaller text.</small>

    游댖The HTML <mark> element defines text that should be marked or highlighted:

        for example:
         
            <p>Do not forget to buy <mark>milk</mark> today.</p>

游리 HTML <del> and <ins> Element

    游댖The HTML <del> element defines text that has been deleted 
        from a document. Browsers will usually strike a line through 
        deleted text:

        for example:

            <p>My favorite color is <del>blue</del> red.</p>

    游댖The HTML <ins> element defines a text that has been inserted 
        into a document. Browsers will usually underline inserted text:

        for example:

            <p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

游리 HTML <sub> and <sup> Element
    
    游댖The HTML <sub> element defines subscript text. Subscript 
        text appears half a character below the normal line, 
        and is sometimes rendered in a smaller font. Subscript 
        text can be used for chemical formulas, like H2O:

            for example:

                <p>This is <sub>subscripted</sub> text.</p>

    游댖The HTML <sup> element defines superscript text. Superscript 
        text appears half a character above the normal line, and is 
        sometimes rendered in a smaller font. Superscript text can 
        be used for footnotes, like WWW[1]:

            for example:

                <p>This is <sup>superscripted</sup> text.</p>
