游리 What is HTML Forms ?

    An HTML form is used to collect user input. The user input is 
    most often sent to a server for processing.

游리 The <form> Element 

    The HTML <form> element is used to create an HTML form for user input:

    For example:

        <form>
            .
            form elements
            .
        </form>

        The <form> element is a container for different types of input 
        elements, such as: text fields, checkboxes, radio buttons, submit 
        buttons, etc.

游리 The <input> Element

    游댖The HTML <input> element is the most used form element.
    游댖An <input> element can be displayed in many ways, depending on the type attribute.

    Here are some examples:

        <input type="text">	    Displays a single-line text input field
        <input type="radio">	Displays a radio button (for selecting one of many choices)
        <input type="checkbox">	Displays a checkbox (for selecting zero or more of many choices)
        <input type="submit">	Displays a submit button (for submitting the form)
        <input type="button">	Displays a clickable button

游리 Text Fields

    The <input type="text"> defines a single-line input field for text input.

    For example:

        <form action="data.html">
            <label for="username">Username</label><br>
            <input type="text" name="username"><br>
            <label for="password">Password</label><br>
            <input type="text" name="password"><br><br>
            <input type="submit" value="Submit">
        </form> 

游리 Radio Buttons

    游댖The <input type="radio"> defines a radio button.
    游댖Radio buttons let a user select ONE of a limited number of choices.

    For example:

        <p>Choose your favorite Web language:</p>

        <form>
            <input type="radio" id="html" name="fav_language" value="HTML">
            <label for="html">HTML</label><br>
            <input type="radio" id="css" name="fav_language" value="CSS">
            <label for="css">CSS</label><br>
            <input type="radio" id="javascript" name="fav_language" value="JavaScript">
            <label for="javascript">JavaScript</label>
        </form>

游리 Checkboxes

    游댖The <input type="checkbox"> defines a checkbox.
    游댖Checkboxes let a user select ZERO or MORE options of a limited number of choices.

    For example:

        <form>
            <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
            <label for="vehicle1"> I have a bike</label><br>
            <input type="checkbox" id="vehicle2" name="vehicle2" value="Car">
            <label for="vehicle2"> I have a car</label><br>
            <input type="checkbox" id="vehicle3" name="vehicle3" value="Boat">
            <label for="vehicle3"> I have a boat</label>
        </form>

游리 The <label> Element

    游댖Notice the use of the <label> element in the example above.
    游댖The <label> tag defines a label for many form elements.
    游댖The <label> element is useful for screen-reader users, because the 
    screen-reader will read out loud the label when the user focuses on 
    the input element.
    游댖The <label> element also helps users who have difficulty clicking 
    on very small regions (such as radio buttons or checkboxes) - because 
    when the user clicks the text within the <label> element, it toggles 
    the radio button/checkbox.
    游댖The for attribute of the <label> tag should be equal to the id 
    attribute of the <input> element to bind them together.

游리 The Submit Button

    游댖The <input type="submit"> defines a button for submitting the form data to a form-handler.
    游댖The form-handler is typically a file on the server with a script for processing input data.
    游댖The form-handler is specified in the form's action attribute.

    For example:

        <form action="data.html">
            <label for="username">Username</label><br>
            <input type="text"><br>
            <label for="password">Password</label><br>
            <input type="text"><br><br>
            <input type="submit" value="Submit">
        </form> 

游리 The Name Attribute for <input>

    游댖Notice that each input field must have a name attribute to be submitted.
    游댖If the name attribute is omitted, the value of the input field will not be sent at all.

    For example:

        <form action="data.html">
            <label for="username">Username</label><br>
            <input type="text" name="username"><br>
            <label for="password">Password</label><br>
            <input type="text" name="password"><br><br>
            <input type="submit" value="Submit">
        </form>
