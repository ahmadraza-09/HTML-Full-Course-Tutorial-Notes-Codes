游리 What is HTML Colors ?

    HTML colors are specified with predefined color names, or with RGB, 
    HEX, HSL, RGBA, or HSLA values.

游리 Color Names

    In HTML, a color can be specified by using a color name:

        游댖Red
        游댖Blue
        游댖Green
        游댖Yellow
        游댖White 
        游댖Purple
        游댖Pink

        游댮 Note: HTML supports 140 standard color names.

游리 Background Color

    You can set the background color for HTML elements:

        For example:

            <h1 style="background-color:DodgerBlue;">Hello World</h1>
            <p style="background-color:Tomato;">Lorem ipsum...</p>  

游리 Text Color
    
    You can set the color of text:

        For example:

            <h1 style="color:Tomato;">Hello World</h1>
            <p style="color:DodgerBlue;">Lorem ipsum...</p>
            <p style="color:MediumSeaGreen;">Ut wisi enim...</p>

游리 Border Color
    
    You can set the color of borders:

        For example:

            <h1 style="border:2px solid Tomato;">Hello World</h1>
            <h1 style="border:2px solid DodgerBlue;">Hello World</h1>
            <h1 style="border:2px solid Violet;">Hello World</h1>

游리 Color Values
    
    游댖In HTML, colors can also be specified using RGB values, HEX values, 
        HSL values, RGBA values, and HSLA values.
    游댖The following three <div> elements have their background color set 
        with RGB, HEX, and HSL values:

        For example:

            1. rgb(255, 99, 71)
            2. #ff6347
            3. hsl(9, 100%, 64%)

        The following two <div> elements have their background color set 
        with RGBA and HSLA values, which add an Alpha channel to the 
        color (here we have 50% transparency):

            1. rgba(255, 99, 71, 0.5)
            2. hsla(9, 100%, 64%, 0.5)


游리 What is HTML Styles - CSS ?

    游댖CSS stands for Cascading Style Sheets.
    游댖CSS saves a lot of work. It can control the layout of multiple web 
        pages all at once.
    游댖With CSS, you can control the color, font, the size of text, the
     spacing between elements, how elements are positioned and laid out, 
     what background images or background colors are to be used, different 
     displays for different devices and screen sizes, and much more!

    游댮Tip: The word cascading means that a style applied to a parent element 
        will also apply to all children elements within the parent. 
        So, if you set the color of the body text to "blue", all headings,
        paragraphs, and other text elements within the body will also get 
        the same color (unless you specify something else)!

游리 How to Use CSS ?

    CSS can be added to HTML documents in 3 ways:

    游댖Inline - by using the style attribute inside HTML elements
    游댖Internal - by using a <style> element in the <head> section
    游댖External - by using a <link> element to link to an external CSS file
    
    The most common way to add CSS, is to keep the styles in external CSS 
    files. However, in this tutorial we will use inline and internal styles, 
    because this is easier to demonstrate, and easier for you to try it 
    yourself.

游리 Inline CSS

    游댖An inline CSS is used to apply a unique style to a single HTML element.
    游댖An inline CSS uses the style attribute of an HTML element.
    游댖The following example sets the text color of the <h1> element to blue, 
        and the text color of the <p> element to red:

        For example:

            <h1 style="color:blue;">A Blue Heading</h1>
            <p style="color:red;">A red paragraph.</p>

游리 Internal CSS

    游댖An internal CSS is used to define a style for a single HTML page.
    游댖An internal CSS is defined in the <head> section of an HTML page, 
        within a <style> element.
    游댖The following example sets the text color of ALL the <h1> elements 
        (on that page) to blue, and the text color of ALL the <p> elements to 
        red. In addition, the page will be displayed with a "powderblue" 
        background color: 

        For example:

            <!DOCTYPE html>
            <html>
            <head>
            <style>
            body {
                background-color: powderblue;
            }
            h1   {
                color: blue;
            }
            p    {
                color: red;
            }
            </style>
            </head>
            <body>

            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>

            </body>
            </html>

游리 External CSS

    游댖An external style sheet is used to define the style for many HTML pages.
    游댖To use an external style sheet, add a link to it in the <head> 
        section of each HTML page:

        For example:

            <!DOCTYPE html>
            <html>
            <head>
            <link rel="stylesheet" href="styles.css">
            </head>
            <body>

            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>

            </body>
            </html>

游리 CSS Colors, Fonts and Sizes

    游댖Here, we will demonstrate some commonly used CSS properties. 
        You will learn more about them later.
    游댖The CSS color property defines the text color to be used.
    游댖The CSS font-family property defines the font to be used.
    游댖The CSS font-size property defines the text size to be used.

        For example:

            <!DOCTYPE html>
            <html>
            <head>
            <style>
            h1 {
            color: blue;
            font-family: verdana;
            font-size: 300%;
            }
            p {
            color: red;
            font-family: courier;
            font-size: 160%;
            }
            </style>
            </head>
            <body>

            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>

            </body>
            </html>

游리 CSS Border

    游댖The CSS border property defines a border around an HTML element.
    游댮Tip: You can define a border for nearly all HTML elements.

        For example:

            p {
                border: 2px solid powderblue;
            }

游리 CSS Padding
    
    The CSS padding property defines a padding (space) between the text and the border.

        For example:

            p {
                border: 2px solid powderblue;
                padding: 30px;
            }

游리 CSS Margin
    
    The CSS margin property defines a margin (space) outside the border.

        For example:

            p {
                border: 2px solid powderblue;
                margin: 50px;
            }
