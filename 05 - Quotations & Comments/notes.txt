游리 What is HTML Quotation and Citation Elements ?

    游댖The HTML <blockquote> element defines a section that is quoted from another source.
    游댖Browsers usually indent <blockquote> elements.

        For example:

            <p>Here is a quote from WWF's website:</p>
            <blockquote cite="http://www.worldwildlife.org/who/index.html">
            For 60 years, WWF has worked to help people and nature thrive. 
            As the world's leading conservation organization, WWF works in
            nearly 100 countries. At every level, we collaborate with 
            people around the world to develop and deliver innovative 
            solutions that protect communities, wildlife, and the places 
            in which they live.
            </blockquote>

游리 HTML <q> for Short Quotations

    游댖The HTML <q> tag defines a short quotation.
    游댖Browsers normally insert quotation marks around the quotation.

        For example:

            <p>
                WWF's goal is to: <q>Build a future where people live in 
                harmony with nature.</q>
            </p>

游리 HTML <abbr> for Abbreviations

    游댖The HTML <abbr> tag defines an abbreviation or an acronym, 
        like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
    游댖Marking abbreviations can give useful information to browsers, 
        translation systems and search-engines.
    游댮Tip: Use the global title attribute to show the description for 
        the abbreviation/acronym when you mouse over the element. 

        for example:
            
            <p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

游리 HTML <address> for Contact Information 

    游댖The HTML <address> tag defines the contact information for the 
        author/owner of a document or an article.
    游댖The contact information can be an email address, URL, physical 
        address, phone number, social media handle, etc.
    游댖The text in the <address> element usually renders in italic, and 
        browsers will always add a line break before and after the <address> element.

        for example:

            <address>
            Written by John Doe.<br>
            Visit us at:<br>
            Example.com<br>
            Box 564, Disneyland<br>
            USA
            </address>

游리 HTML <cite> for Work Title 

    游댖The HTML <cite> tag defines the title of a creative work 
        (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
    游댮Note: A person's name is not the title of a work.
    游댖The text in the <cite> element usually renders in italic

        for example:

            <p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.</p>

游리 HTML <bdo> for Bi-Directional Override

    游댖BDO stands for Bi-Directional Override.
    游댖The HTML <bdo> tag is used to override the current text direction:

        for example:

            <bdo dir="rtl">This text will be written from right to left</bdo>


游리 What is HTML Comments ?

    HTML comments are not displayed in the browser, but they can help document your HTML source code.

游리 HTML Comment Tag
    
    You can add comments to your HTML source by using the following syntax:

        for example:

            <!-- Write your comments here -->

游리 Add Comments
    
    With comments you can place notifications and reminders in your HTML code:

        for example:

            <!-- This is a comment -->
            <p>This is a paragraph.</p>
            <!-- Remember to add more information here -->

游리 Hide Content
    
    游댖Comments can be used to hide content.
    游댖This can be helpful if you hide content temporarily:

        for example:

            <p>This is a paragraph.</p>
            <!-- <p>This is another paragraph </p> -->
            <p>This is a paragraph too.</p>

            游댮You can also hide more than one line. Everything between the 
                <!-- and the --> will be hidden from the display.

            <p>This is a paragraph.</p>
            <!--
            <p>Look at this cool image:</p>
            <img border="0" src="pic_trulli.jpg" alt="Trulli">
            -->
            <p>This is a paragraph too.</p>

游리 Hide Inline Content 

    Comments can be used to hide parts in the middle of the HTML code.

        for example:

            <p>This <!-- great text --> is a paragraph.</p>
